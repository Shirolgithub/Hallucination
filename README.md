# TruthShield-RAG

TruthShield-RAG is an AI-powered hallucination detection framework built using Retrieval-Augmented Generation (RAG), FAISS vector search, and Large Language Models (LLMs).

The project detects hallucinated responses generated by language models and improves answer reliability using semantic retrieval and confidence scoring.

---

# Features

- Hallucination Detection
- Retrieval-Augmented Generation (RAG)
- Semantic Search using FAISS
- Confidence Scoring
- Severity Classification
- Flask Web Application
- Accuracy Evaluation Dashboard
- Real-time Response Correction

# Installation

## Clone Repository

```bash
git clone https://github.com/Shirolgithub/Hallucination.git
```

## Move Into Project Folder

```bash
cd Hallucination
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run Project

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
