# TruthShield-RAG

TruthShield-RAG is a GenAI framework for detecting and correcting hallucinations in LLM-generated responses using Retrieval-Augmented Generation (RAG).

## Features

- Hallucination Detection
- Confidence Scoring
- Retrieval-Augmented Generation
- Corrected Answer Generation
- Accuracy Evaluation
- Flask Web Interface

## Tech Stack

- Python
- Flask
- HuggingFace Transformers
- Scikit-learn
- Matplotlib

## Run Project

```bash
pip install -r requirements.txt
python app.py
